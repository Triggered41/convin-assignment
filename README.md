# convin-assignment

## Setup
Install all the packages using the command (You may use a virtual environment if you want)

1. `python -m pip install -r requirements.txt`
2. `cd expenses_site`

## Run Project

`python manager.py runserver`


## Api Endpoints

1. user_login
2. create_user
3. get_user
4. add_expense
5. get_expenses
6. get_all_expenses
7. get_overall_expense
8. get_balance_sheet
